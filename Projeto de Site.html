História da web

Primeiros computadores

(Voltar)

A 1° geração de computadores foi o Harvard Mark I (1944) era um computador eletromecânico que era basicamente uma máquina de cálculos. logo após surgiu o Colossus (1946) que também era destinado a cálculos, criptoanálises pela Alemanha, que foi seu criador. E em 1946 surgiu o ENIAC pelos EUA que já era mais automatizado.

A 2° geração de computadores aconteceu entre os anos (1959 a 1965) que foi a evolução das válvulas utilizadas na primeira geração, para os transistores que foi de grande ajuda para diminuir o tamanho dos computadores.

A 3° geração de computadores de (1965 a 1970) funcionavam com circuitos integrados. Eles foram substituídos pelos transistores. A partir daí apresentavam uma dimensão menor e capacidade maior de processamento. Sendo assim foram criados os chips e a utilização de computadores pessoais teve início.

A 4° geração de computadores de (1971 até hoje) é caracterizada pelo aperfeiçoamento da tecnologia já existente, proporcionando uma otimização da máquina para os problemas do usuário, maior grau de miniaturizção, confiabilidade e velocidade maior, já na ordem e nanosegundos.

Como surgiu

(Voltar)

Um dos primeiros meios de comunicação foi o telégrafo. Em 1958 foi traçado um cabo de um continente a outro que fez com que a comunicação intercontinental fosse possível pela primeira vez.

Na década de 1950 houve a Guerra Fria entre os EUA e a União Soviética sobre a briga da corrida espacial, sendo assim, naquela época houve muitas inovações tecnológicas.

Após muitas evoluções surgiu a ARPANET, que a partir dela surgiram sistemas de comunicações em pacotes enviados um a um e que permitiu múltiplos destinos. Pontos de intersecção de informações funcionando como "checkpoints". Poderíamos dizer que foi o avô dos roteadores.

A primeira conexão estabelecida foi feita em (29/10/1969) entre a UCLA (University of California) e SRI (Stanford Research Institute). Porém no mesmo dia houve a primeira queda de conexão.

E em (1989), Tim Berneers-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar o WorldWideWeb(WWW), o primeiro navegador chamado WorldWideWeb.

Sobre servidores

(Voltar)

Linguagens de Programação

O que é?

(Voltar)

É uma linguagem formal que especifica um conjunto de instruções e regras usadas para gerar programas (Softwares).

Server-side

(Voltar)

É a lingueagem que é executada no servidor, também conhecida como back-end. É o termo usado para designar operações que, em um contexto cliente-servidor, são feitas no servidor, não no cliente.

Existem várias razões para isso: desde o meio poder de processamento do servidor, até a presença de softwares ou recursos que estejam disponíveis apenas no servidor.

Client-side

(Voltar)

É a linguagem que é executada no lado Client, também conhecido como front-end refere-se às operações que são realizadas pelo cliente em um relacionamento cliente-servidor em uma rede de computadores.

HTML não é linguagem de Programação

(Voltar)

NÃO! HTML não é uma lingueagem de programação, mas sim um arquivo de marcação. Textos delimitados por nomes que o navegador consegue interpretar e cada nome destes delimitadores, tem uma função e comportamento específico.

Inspetor de elementos

(Voltar)

Inspecionar elemento é um utilitário que permite visualizar o código-fonte subjacente de qualquer página web. Além disso, você pode usá-la para fazer alterações temporárias e ver os resultados em tempo real, deixando o código-fonte original intacto. Utilizando o Chrome como exemplo, clicando F12 você terá acesso ao inspetor de elementos.

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Listas Ordenadas e Não Ordenadas

(Voltar)

Nas listas Ordenadas você utiliza a tag "ol" e a "li" para definir um item da lista.

Nas listas não Ordenadas você utiliza a tag "ul".

A lista ordenada é feita em sequência de números como: 1, 2, 3, etc. Enquanto a não ordenada é feita com marcações como: •.

Links

(Voltar)

O Elemento HTML link especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.
